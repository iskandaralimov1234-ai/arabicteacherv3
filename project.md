# Madinah Arabic Course Teacher (Pro Max)

## Описание проекта
Интерактивное веб-приложение для изучения арабского языка по всемирно известному "Мединскому курсу" (Madinah Arabic Course). Проект предлагает структурированные уроки с грамматикой, словарем и упражнениями, адаптированными под высокие стандарты качества (Pro Max).

## Основные характеристики
- **Многоязычность**: Поддержка уроков на разных языках (английский, русский, узбекский).
- **Высокий стандарт качества (Pro Max)**: 
  - Использование правильной арабской терминологии (Нахв: Marfū', Manṣūb, Majrūr, Majzūm).
  - Строгое соблюдение структуры уроков (минимум 4 секции теории, 10 тестов: 4 грамматика, 3 перевод, 3 словарь).
  - Огласовки (харакяты) и транслитерация для всех арабских текстов.
- **Интерактивные упражнения**: Множественный выбор, сопоставление, сборка предложений.
- **Аудио-сопровождение**: Возможность прослушивания произношения.

## Технологический стек
- React/Next.js (фронтенд)
- TypeScript (строгая типизация данных уроков)
- Tailwind CSS (стилизация)
- Zustand/Redux (управление состоянием)

## Структура данных уроков
Каждый урок содержит:
1. `title` - Название урока
2. `grammar` - Основная грамматическая тема
3. `theory` - Теоретический блок (Описание, Правила, Примеры, Словарь)
4. `tasks` - Массив тестов для проверки знаний

## Рекомендации по разработке
При добавлении новых уроков обязательно сверяться со стандартом качества `Madina Arabic (Pro Max)`, описанным в `.agent/workflows/madina-quality-standard.md`.
